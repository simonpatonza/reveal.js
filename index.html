  • Legacy

"Business Service Bus"

— CBDI

What is this?

Logical view of the available and used services for a particular business domain

Why?

Guides developers to a set of appropriate services for their domain

How?

Services on this bus should have the same common specifications, policies, etc
Eg: standards, security policies, part of the same business domain
